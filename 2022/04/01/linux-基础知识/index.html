vim是vi的升级

vim有三种模式：普通模式、插入模式、命令模式

< !– more –>

  1. 普通模式->插入模式：

    • i 光标不动，在原光标的前面插入数据
    • a光标后移一位，在光标后面插入数据
    • I在本行的开始位置插入数据
    • A在本行的结束位置处插入数据
    • o 在本行下面创建一个空行
    • O 在本行的上一行创建一个空行

  2. 普通模式下的命令较多

  3. 命令模式下的命令：

    • wq 保存并退出文件
    • q 退出文件
    • q! 强制退出

查看文件的方式

  1. cat [-n -b ] filename
  2. tac 倒叙查看文件内容 （从最后一行开始）
  3. head [-n] filename直接查看,默认前10行
  4. tail [-n] filename 倒叙查看，默认前10行
  5. more filename 文件过大，分页显示内容，空格建翻页
  6. wc 文件名 统计一个文件中各种数据的数量计算文件的byte 数，字数，列数等
